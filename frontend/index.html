Top P
Nucleus sampling. Samples from the smallest set of tokens whose cumulative probability ≥ top_p. Lower values restrict vocabulary to the most likely tokens.
Claude does not expose top_p directly. The vocabulary effect is simulated via system instruction calibrated to the chosen value.
